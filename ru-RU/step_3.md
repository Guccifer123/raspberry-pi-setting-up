## Настройка microSD-карты

Если у тебя есть microSD-карта, на которой еще нет операционной системы Raspbian или если ты хочешь отменить все сделанные тобой изменения в файлах операционной системы своей Raspberry Pi, то ты можешь легко установить Raspbian самостоятельно. Для этого тебе нужен компьютер с портом для microSD-карты - у большинства ноутбуков и настольных компьютеров есть такой. В противном случае, ты можешь использовать внешнее устройство для чтения microSD-карт.

### Установка операционной системы Raspbian через NOOBS

Использование программного обеспечения NOOBS - это самый простой способ установки Raspbian на microSD-карту.

#### Загрузка NOOBS

+ Посети [страницу загрузок Raspberry Pi](https://www.raspberrypi.org/downloads).

![Страница загрузки](images/downloads-page.png)

+ Должно появиться окно с кнопкой-ссылкой на NOOBS. Щёлкни по этой кнопке-ссылке.

![Нажми на NOOBS](images/click-noobs.png)

+ Самый простой вариант - загрузить zip-архив. Обязательно обрати внимание на то, куда ты сохраняешь архив, чтобы его можно было быстро найти.

![Загрузить zip](images/download-zip.png)

#### Форматирование microSD-карты

Все, что хранится на microSD-карте, будет перезаписано во время форматирования. Поэтому, если на microSD-карте, на которую ты хочешь установить Raspbian, в настоящее время есть какие-либо файлы, например, от старой версии Raspbian, то перед форматированием обязательно скопируй их в безопасное место, чтобы не потерять их навсегда.

+ Посети веб-сайт SD Association и загрузи [SD Formatter](https://www.sdcard.org/downloads/formatter_4/index.html) для Windows или Mac.

+ Следуй инструкциям по установке этой программы.

+ Вставь microSD-карту в подходящий для неё разъём компьютера или ноутбука и обрати внимание на выделенную им букву диска, например `F:`.

+ В SD Formatter выбери букву диска для microSD-карты и запусти процесс форматирования.

#### Извлечение NOOBS из zip-архива

Теперь тебе нужно будет извлечь файлы из zip-архива NOOBS, загруженного с веб-сайта Raspberry Pi.

+ Найди загруженный архив - по умолчанию он должен находиться в папке `Загрузки`.

+ Для извлечения файлов из архива, примени к нему двойной щелчок и оставь открытым это окно Проводника (Windows) или Finder (MacOS).

#### Копирование файлов

+ Теперь открой другое окно Проводника (Windows) или Finder (MacOS) и перейди к microSD-карте. Лучше всего расположить эти два окна бок о бок.

+ Выбери все файлы из папки `NOOBS` и перетащи их в окно microSD-карты, чтобы скопировать их на карту.

![копирование в Windows](images/copy3.png)

![копирование в macos](images/macos_copy.png)

+ Как только файлы будут скопированы, ты можешь извлечь microSD-карту.
